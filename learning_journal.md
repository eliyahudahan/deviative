## 📚 סיכום – Docker, PostgreSQL, Python, Git

---

### 🐳 Docker

| מושג | הסבר | אנלוגיה |
|-------|--------|----------|
| **Image** | תבנית מוכנה – `postgres:15`. סטטית. לא רצה. | תערובת עוגה בקופסה |
| **Container** | Image במצב פעיל. רץ, צורך זיכרון. | העוגה שיצאה מהתבנית |
| **`docker run`** | לוקח Image – יוצר Container – מריץ. | לפתוח קופסה, לאפות, להגיש |
| **`-d`** | רץ ברקע – לא תופס את החלון. | תנור שעובד בזמן שאתה בסלון |
| **`-p 5432:5432`** | גשר – פותח פורט בין המחשב לקונטיינר. | דלת בין המטבח לסלון |
| **`-e`** | משתני סביבה – משתמש, סיסמה, מסד נתונים. | תווית על הקופסה |

### 🔌 Socket

| מושג | הסבר | אנלוגיה |
|-------|--------|----------|
| **Socket** | צינור תקשורת וירטואלי בין שתי תוכנות. | קו טלפון |
| **Unix Socket** | קובץ – PostgreSQL יוצר אוטומטית כשהוא רץ. | טלפון נייד של PostgreSQL |
| **TCP Socket** | חיבור רשת – Python יוצר דרך `psycopg2`. | טלפון נייד של Python |
| **Port 5432** | המספר שדרכו Python ו-PostgreSQL מדברים. | מספר טלפון |
| **Docker `-p`** | גשר – מחבר TCP (חיצוני) ל-Unix Socket (פנימי). | מרכזייה |

### 🐘 PostgreSQL

| מושג | הסבר |
|-------|--------|
| **PostgreSQL** | מסד נתונים – מאחסן מידע בטבלאות. מאזין דרך Unix Socket. |
| **psycopg2** | ספריית Python – מתווך. מתרגם פקודות Python ← SQL. |

| **`SERIAL`** | אוטו-אינקרמנט – מספר עולה אוטומטית. |

### 📊 תרשים – שליחת פקודת SQL

```
Python
    │ cursor.execute("CREATE TABLE...")
    ▼
psycopg2 ← 📞 TCP Socket
    │
Docker -p 5432:5432 ← 📞 מרכזייה
    │
Container ← PostgreSQL רץ
    │
Unix Socket ← 📱 PostgreSQL מאזין
    │
PostgreSQL מעבד – יוצר טבלה
    │
תשובה: "Table created"
    │
psycopg2 ← 📞 חוזר ל-Python
    │
print("Table created")
```

### 🎯 אנלוגיית המטבח – כל התהליך

| שלב | אנלוגיה |
|------|----------|
| **Image `postgres:15`** | קופסת תערובת עוגה |
| **`docker run`** | פותח קופסה, שם בתנור |
| **Container** | עוגה בתנור – PostgreSQL רץ |
| **`-p 5432:5432`** | דלת תנור פתוחה |
| **`psycopg2.connect()`** | אתה ניגש לתנור |
| **`cursor.execute()`** | אתה מבקש: "תכין לי פרוסה" |
| **PostgreSQL מעבד** | האופה חותך – מכין |
| **תשובה** | הפרוסה בצלחת |

### 📞 אנלוגיית הבנק – שליחת פקודה

| שלב | אנלוגיה |
